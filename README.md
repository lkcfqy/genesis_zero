# Genesis: Artificial Life & Neuroevolution Simulation
# 创世纪：人工生命与神经进化模拟

**Genesis** 是一个探索机器智能起源的实验性项目。通过 6 个渐进的进化阶段，模拟了智能体（Agent）如何从最基本的逻辑运算单元，一步步进化出空间感知、记忆能力、博弈策略，最终形成具备社会分工与部落合作的高级群体。

本项目展示了**神经进化（Neuroevolution）**与**人工生命（Artificial Life）**的核心魅力：智能不是被设计出来的，而是在生存压力下涌现的。

---

## 📂 项目结构与进化阶段 (Stages)

该项目包含 6 个核心脚本，代表了智能进化的不同里程碑：

### Stage 0: 智能的火花 (The Spark)
- **文件**: `main.py`
- **目标**: 解决 XOR（异或）逻辑问题。
- **核心机制**: **自我重构 (Self-Modifying Architecture)**。
  - 初始神经网络是一个极简的线性感知机，从数学上无法解决 XOR 问题。
  - 当网络陷入瓶颈（Loss 停滞）时，它会自动“动手术”——在架构中插入隐藏层或增加神经元。
  - **看点**: 观察机器如何在“学不会”的压力下，物理生长出能理解复杂逻辑的大脑。

### Stage 1: 神经网络社会 (Neural Society)
- **文件**: `genesis_one.py`
- **目标**: 种群在逻辑世界中的优胜劣汰。
- **核心机制**: **自然选择 (Natural Selection)**。
  - 引入 `Organism`（生物体）概念，不再是单体进化。
  - 只有解出 XOR 难题的个体才能获得能量并繁衍，将大脑结构遗传给下一代。
  - 包含可视化图表生成，展示种群数量与智力水平（Loss）的协同演化。

### Stage 2: 狩猎场 (Hunting Grounds)
- **文件**: `genesis_two.py`
- **目标**: 空间感知与觅食。
- **环境**: 2D 栅格世界 (Pygame)。
- **核心机制**: **具身智能 (Embodied Intelligence)**。
  - 智能体拥有 `HunterBrain`，输入视觉信号（食物坐标），输出动作（移动）。
  - 必须学会移动去寻找食物维持生命。
  - 只有活得够久的个体才有机会发生大脑结构的突变。

### Stage 3: 黑暗森林 (Dark Forest)
- **文件**: `genesis_three.py`
- **目标**: 记忆与博弈。
- **核心机制**: **循环神经网络 (RNN/GRU)**。
  - 引入 `RecurrentBrain`，智能体拥有了“记忆”，能处理序列信息。
  - **博弈**: 视野中包含墙壁、食物、强者（威胁）和弱者（猎物）。
  - 智能体必须在捕食同类和躲避强者之间做出决策，模拟了残酷的生存竞争。

### Stage 4: 自律进化 (Autonomous Evolution)
- **文件**: `genesis_four.py`
- **目标**: 自我设计与环境平衡。
- **核心机制**: **动态环境与大脑可塑性**。
  - **EvolvingBrain**: 智能体可以自主决定进化方向——是“看得更清”（加宽视觉层）还是“记得更多”（增加 GRU 记忆容量）。
  - **自动平衡仪 (Auto-Balancer)**: 环境会根据人口密度自动调节食物生成率，维持生态系统的动态平衡。

### Stage 5: 部落战争 (Tribal Wars)
- **文件**: `genesis_five.py`
- **目标**: 沟通与合作。
- **核心机制**: **社会性 (Social Behavior)**。
  - **Tribes**: 智能体分为红蓝两族，拥有视觉识别能力。
  - **Cooperation**: 地图中出现了强大的猎物（猛犸象），单体无法击杀。智能体必须学会识别同族队友，通过“语言”（信号输出）协调，共同围猎才能生存。
